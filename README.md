# Student Grade Calculator

## Description
This is a basic C# Windows Forms application designed to calculate the average grade of two numbers (representing student scores) and display appropriate messages based on the calculated average.

## Features
- Calculates the average of two input numbers.
- Displays messages indicating whether the student passed, received a commendation, or failed based on the average score.
- Provides a clear button to reset the input fields.

## Installation
1. Ensure you have Visual Studio installed.
2. Clone this repository to your local machine.
3. Open the solution file in Visual Studio.
4. Build and run the project.

## Usage
1. Enter two numbers in the text boxes.
2. Click the 'Kontrol' (Check) button to calculate the average and display the message.
3. Use the 'Temizle' (Clear) button to reset the input fields.

## Tech Stack
- C#
- Windows Forms
